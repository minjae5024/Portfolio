PROJECTS

농산물 가격 조회 및 커뮤니티

공공데이터 API를 활용하여 전국의 농산물 도매 가격 정보를 조회하고, 사용자들끼리 소통할 수 있는 커뮤니티 서비스입니다.

주요 기능

  • 공공데이터 API 연동 및 데이터베이스 저장
  • 지도 API 기반 시장 선택 및 가격 조회
  • 시장별 사용자 커뮤니티 게시판 (CRUD)
  • 조건 동적 검색 기능(QueryDSL)

적용 기술

JAVA Spring Boot JPA QueryDSL MySQL AWS EC2/RDS GitHub Actions
  • 데이터베이스 최적화 : 게시글 조회 시 발생한 N+1 문제를 발견하고 Fetch Join을 적용하여 API 응답 속도를 개선했으며 users테이블에 인덱스를 적용했습니다.
  • 개발 생산성 향상 : GitHub Actions를 활용해 테스트, 빌드, 배포 자동화(CI/CD) 파이프라인을 구축했습니다.
  • 민감정보 관리 : 외부 API키, JWT 시크릿 키 등의 민감정보를 GitHub Secrets와 환경변수를 이용해 외부로 분리했습니다.

이커머스 API 서버

사용자 인증, 상품 관리, 장바구니, 주문, 결제 등 실제 서비스의 확장성과 안정성을 목표로 설계한 이커머스 API 서버입니다.

주요 기능

  • 동시성 제어를 적용한 재고 관리 시스템
  • JWT & Spring Security 기반 인증/인가
  • 관리자(Admin) 권한 분리를 통한 상품 관리
  • 사용자별 장바구니 및 주문/결제 기능 구현

적용 기술

JAVA Spring Boot JPA MySQL AWS EC2/RDS GitHub Actions
  • 동시성 제어 : 다수의 동시 주문 상황에서 발생할 수 있는 재고 데이터의 불일치 문제를 해결하기 위해 비관적 락(Pessimistic Lock)을 적용했습니다.
  • 데이터 무결성 : `@Transactional`을 적용하여 주문/결제 프로세스에서 데이터의 일관성을 보장했습니다.
  • CI/CD : GitHub Actions를 이용한 CI/CD 파이프라인을 설계하고 구현했습니다.
