PROJECTS

이커머스 API 서버

사용자 인증, 상품 관리, 장바구니, 주문, 결제 등 실제 운영 및 확장성을 고려하여 설계한 이커머스 API 서버입니다.

주요 기능

  • JWT & Spring Security 기반 사용자 인증
  • Admin 권한 분리를 통한 상품 관리 (CRUD)
  • 사용자별 장바구니 및 주문 기능
  • 내부 포인트를 이용한 결제 시뮬레이션

적용 기술

JAVA Spring Boot JPA MySQL AWS RDS EC2
  • 객체지향 설계: 도메인 객체가 핵심 비즈니스 로직을 갖도록 설계하여 응집도를 높이고 서비스 계층의 복잡도를 낮췄습니다.
  • DTO 계층 분리: API 스펙의 안정성을 확보하고 엔티티를 보호하기 위해 DTO를 적극적으로 활용했습니다.
  • 데이터 무결성: `@Transactional`을 적용하여 주문/결제 프로세스에서 데이터의 일관성을 보장했습니다.

농산물 가격 조회 및 커뮤니티

공공데이터 API를 활용하여 전국의 농산물 도매 가격 정보를 조회하고, 사용자들끼리 소통할 수 있는 커뮤니티 서비스입니다.

주요 기능

  • 공공데이터 API 연동 및 데이터베이스 저장
  • 지도 기반 시장 선택 및 가격 조회
  • QueryDSL을 이용한 동적 조건 검색
  • 시장별 사용자 커뮤니티 게시판

적용 기술

JAVA Spring Boot QueryDSL MySQL GitHub Actions
  • 성능 최적화: N+1 문제를 직접 발견하고 페치 조인(Fetch Join)으로 해결하여 API 응답 속도를 개선했습니다.
  • 인프라 개선: EC2 메모리 부족 문제를 로그 분석으로 진단하고, AWS RDS로 데이터베이스를 분리하여 해결했습니다.
  • CI/CD 구축: GitHub Actions를 이용해 테스트, 빌드, 배포 자동화 파이프라인을 구축하여 개발 생산성을 높였습니다.
