소개

안녕하세요. 사용자의 요구사항을 안정적인 코드로 구현하고, 비즈니스의 성장에 기여하는 백엔드 개발자 이민재입니다. Java와 Spring Boot를 기반으로 한 웹 애플리케이션 개발에 깊은 흥미를 느끼고 있습니다.

단순히 기능을 구현하는 것을 넘어, 대용량 트래픽 처리, 시스템 아키텍처 설계, 코드 품질 개선 등 기술적인 도전을 즐깁니다. 최근 정보처리기사 자격증을 취득하며 CS 기본 지식을 공고히 다졌으며, 이를 바탕으로 더 깊이 있는 개발자가 되기 위해 꾸준히 학습하고 있습니다.

팀원들과의 원활한 소통과 협업을 중요하게 생각하며, 기술 블로그 운영을 통해 학습한 내용을 기록하고 공유하는 것을 좋아합니다.

기술 스택

핵심 기술 (Core)

  • Java - 객체지향 프로그래밍에 대한 깊은 이해
  • Spring Boot - RESTful API 개발 및 MVC 패턴 활용
  • JPA / Hibernate - ORM을 활용한 데이터 영속성 관리
  • SQL - MySQL, PostgreSQL 기반의 쿼리 작성 및 최적화
  • Spring Security - JWT 기반 인증/인가 구현

심화/우대 기술 (Advanced)

  • Redis - 캐싱을 통한 성능 개선 경험
  • Kafka - 비동기 메시지 큐를 이용한 시스템 설계
  • Docker - 컨테이너 기반 애플리케이션 배포 경험
  • AWS - EC2, RDS, S3 등 기본 서비스 사용 경험

도구 및 협업 (Tools)

  • Git & GitHub - 버전 관리 및 협업
  • Jenkins - CI/CD 파이프라인 구축 경험
  • IntelliJ - 주력 개발 IDE
  • Slack / Notion - 원활한 커뮤니케이션 및 문서화

프로젝트

비동기 주문 처리 이커머스

대규모 주문 트래픽을 안정적으로 처리하기 위한 백엔드 시스템

Spring Boot Kafka Redis JPA Docker
  • Kafka 메시지 큐를 도입하여 주문 요청을 비동기적으로 처리, 시스템 부하 분산 및 안정성 확보
  • Redis를 활용한 상품 정보 캐싱으로 API 응답 속도 40% 개선
  • Spring Security와 JWT를 이용한 안전한 인증/인가 시스템 구현
  • Docker를 통해 개발 환경을 컨테이너화하여 배포 일관성 유지

실시간 채팅 애플리케이션

WebSocket 기반의 다자간 실시간 채팅 서비스

Spring Boot WebSocket Redis (Pub/Sub) Docker

  • Redis Pub/Sub 모델을 활용하여 여러 채팅 서버 인스턴스 간 메시지 브로커링 구현, 수평적 확장성 확보
  • 채팅방 생성, 참여, 메시지 전송 등 핵심 기능 API 개발
  • Nginx를 이용한 WebSocket 연결 리버스 프록시 설정
