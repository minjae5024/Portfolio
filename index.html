소개

안녕하세요. 사용자의 요구사항을 안정적인 코드로 구현하고, 비즈니스의 성장에 기여하는 백엔드 개발자 이민재입니다. Java와 Spring Boot를 기반으로 한 웹 애플리케이션 개발에 깊은 흥미를 느끼고 있습니다.

단순히 기능을 구현하는 것을 넘어, 대용량 트래픽 처리, 시스템 아키텍처 설계, 코드 품질 개선 등 기술적인 도전을 즐깁니다. 최근 정보처리기사 자격증을 취득하며 CS 기본 지식을 공고히 다졌으며, 이를 바탕으로 더 깊이 있는 개발자가 되기 위해 꾸준히 학습하고 있습니다.

팀원들과의 원활한 소통과 협업을 중요하게 생각하며, 기술 블로그 운영을 통해 학습한 내용을 기록하고 공유하는 것을 좋아합니다.

기술 스택

백엔드 (Backend)

  • Java - 객체지향 프로그래밍에 대한 깊은 이해
  • Spring Boot - RESTful API 개발 및 MVC 패턴 활용
  • JPA / QueryDSL - ORM 및 동적 쿼리를 활용한 데이터 관리
  • Spring Security - JWT 기반 인증/인가 시스템 구현

데이터베이스 (Database)

  • MySQL - 관계형 데이터베이스 사용 및 스키마 설계
  • SQL - 데이터 조작 및 쿼리 최적화

도구 및 협업 (Tools & DevOps)

  • Git & GitHub - 버전 관리 및 협업
  • Docker - 컨테이너 기반 애플리케이션 배포 경험
  • IntelliJ - 주력 개발 IDE

프로젝트

마켓 프라이스 (농산물 가격 정보 서비스)

공공 API 데이터를 활용한 실시간 농산물 가격 정보 제공 및 커뮤니티 웹 서비스

Spring Boot Spring Security JPA & QueryDSL MySQL Docker
  • 문제 해결 중심 설계: 단순 CRUD를 넘어, 복잡한 검색 및 정렬 요구사항을 해결하기 위해 QueryDSL을 도입하여 동적 쿼리 생성 및 성능 최적화 문제를 해결했습니다.
  • 보안 내재화: Spring Security와 JWT를 적용하여 안정적인 인증/인가 시스템을 구축했습니다. 특히, JWT 비밀키를 환경 변수로 분리하고 XSS 방어 로직을 추가하여 실제 운영 환경 수준의 보안을 고려했습니다.
  • 사용자 중심 기능 구현: 공공 API 데이터를 가공하여 사용자에게 직관적인 지도 기반 가격 정보를 제공하고, 시장별 커뮤니티 기능을 통해 사용자 참여를 유도했습니다.
  • 전역 예외 처리: @RestControllerAdvice를 활용하여 시스템 전반의 예외를 일관되게 처리함으로써 코드 중복을 줄이고 안정성을 높였습니다.
